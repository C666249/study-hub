## 三、JSON 数据格式

每题一个对象，所有题组成数组。**每个字段都是必需的。**

### 3.1 判断题 (type: "judge")

```json
{
  "id": "hw1_judge_1",
  "type": "judge",
  "homework": 1,
  "tags": ["刑法的效力"],
  "question": "外国人在中国领域内犯罪的，都适用我国刑法。",
  "options": null,
  "answer": "错",
  "explanation": "根据《刑法》第11条，享有外交特权和豁免权的外国人犯罪通过外交途径解决，并非一律适用我国刑法。"
}
```

### 3.2 单选题 (type: "single")

```json
{
  "id": "hw2_single_1",
  "type": "single",
  "homework": 2,
  "tags": ["危害行为"],
  "question": "下列哪一选项构成不作为犯罪？",
  "options": {
    "A": "丁邀秦某到风景区漂流…丁未下河救人，秦某溺亡",
    "B": "乙女拒绝周某求爱…乙未呼救，周某溺亡",
    "C": "甲到湖中游泳…甲未予理睬，武某溺亡",
    "D": "丙将不善游泳的贺某拉到深水区…贺某溺亡"
  },
  "answer": "D",
  "explanation": "A、B、C均无刑法上救助义务。D正确：丙的先行行为使其产生救助义务，能救而不救，成立不作为犯罪。"
}
```

### 3.3 多选题 (type: "multiple")

```json
{
  "id": "hw1_multiple_1",
  "type": "multiple",
  "homework": 1,
  "tags": ["刑法的效力"],
  "question": "关于刑事管辖权，下列哪些选项是正确的？",
  "options": {
    "A": "甲在国外教唆陈某到中国境内实施绑架行为，中国司法机关对甲的教唆犯罪有刑事管辖权",
    "B": "隶属于中国某边境城市旅游公司的长途汽车在从中国进入E国境内之后…",
    "C": "中国法院适用普遍管辖原则对劫持航空器的丙行使管辖权时…",
    "D": "外国人丁在中国领域外对中国公民犯罪的…也可能不适用中国刑法"
  },
  "answer": "ABD",
  "explanation": "A正确：属地管辖的扩展。B错误：长途汽车进入外国境内后不视为中国领土延伸。C错误：普遍管辖的定罪量刑依据是中国刑法而非国际条约。D正确：保护管辖有条件限制。"
}
```

### 3.4 简答题 (type: "short")

```json
{
  "id": "hw1_short_1",
  "type": "short",
  "homework": 1,
  "tags": ["刑法的效力"],
  "question": "简答我国刑法对空间效力作了哪些规定？",
  "options": null,
  "answer": "属地管辖原则、属人管辖原则、保护管辖原则、普遍管辖原则。",
  "explanation": "属地管辖（第6条）：凡在中国领域内犯罪的，除法律有特别规定外，都适用本法。属人管辖（第7条）：中国公民在中国领域外犯本法规定之罪的，适用本法。保护管辖（第8条）：外国人在中国领域外对中国国家或公民犯罪…普遍管辖（第9条）：对于中国缔结或参加的国际条约所规定的罪行…"
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 唯一标识，格式：`hw{作业号}_{题型}_{序号}`，如 `hw2_single_3` |
| `type` | string | `"judge"` / `"single"` / `"multiple"` / `"short"` |
| `homework` | int | 作业编号（1, 2, 3…） |
| `tags` | string[] | 知识点标签，如 `["危害行为", "因果关系"]`。从 docx 末尾"知识点：XXX"提取 |
| `question` | string | 题干（纯文本，去掉题号和分值） |
| `options` | object\|null | 选项，key 为 A/B/C/D。判断题和简答题填 `null` |
| `answer` | string | **正确答案**。判断：`"对"`/`"错"`；单选：`"A"`~`"D"`；多选：`"ABD"`（字母升序）；简答：参考要点全文 |
| `explanation` | string | **解析**。如果 docx 里有"答案解析"段落就用它（整理措辞）；如果没有则根据知识点补充 |

## 四、docx 提取方法

### 4.1 原理

docx 本质是 ZIP 文件。解压后，内容在 `word/document.xml` 中。文本在 `<w:t>` 标签内。

### 4.2 提取命令（Bash）

```bash
# 1. 解压
cp "作业一.docx" /tmp/hw1.docx
cd /tmp && unzip -o hw1.docx -d hw1_content

# 2. 提取纯文本
cat hw1_content/word/document.xml \
  | sed 's/<[^>]*>//g' \
  | sed 's/&#[0-9]*;//g' \
  | tr -s ' \t\n' '\n'
```

### 4.3 解析注意事项

**区分正确答案和我的答案**：

docx 中可能包含两种答案标记：
- `正确答案：D` ← 用这个
- `我的答案：B` ← **忽略！**这是学生提交的答案
- `0分` / `1分` ← 得分标记，忽略

提取逻辑：
1. 如果有 `正确答案：X` → 直接用
2. 如果只有 `我的答案：X` 且没有 `正确答案` → 需要你自己判断对错，标记正确答案
3. 如果完全没有答案标记 → 多选题/单选题需要你根据法律知识判断

**从"答案解析"段提取知识点标签**：
- 作业二的 docx 每道题末尾有 `知识点：危害行为的分类`
- 提取这个作为 `tags` 值

## 五、解析补充规则

### 5.1 docx 有解析的情况

作业二每道题有"答案解析"段落（虽然格式混乱和选项混在一起）。提取后需要：
- 清理格式（去重、分段、纠正选项字母映射错位）
- 保持 1-3 句话，简洁明了
- 格式："A错误/正确：原因。B错误/正确：原因。…"

### 5.2 docx 无解析的情况（如作业一、马原）

自行补充，每条解析满足：
- **1-3 句话**，控制在 100 字以内
- **点出关键法条或原则**（如"根据《刑法》第X条…"）
- **解释对/错原因**，不只是"A正确B错误"
- **多选题要逐选项说明**

示例（补充的解析）：

```json
"explanation": "根据《刑法》第6条属地管辖原则，凡在中国领域内犯罪的均适用本法。但第11条规定享有外交特权与豁免的外国人通过外交途径解决。因此并非所有外国人都适用，表述错误。"
```

### 5.3 简答题解析

简答题的 `explanation` 内容与 `answer` 一致即可（更详细的版本），或者补充相关法条。

## 六、输出规范

### 6.1 文件命名

```
data/刑法.json    ← 所有刑法题目
data/马原.json    ← 所有马原题目
data/行政法.json   ← 所有行政法题目
```

**每个科目一个 JSON 文件，包含该科目所有作业的题目。** 不要按作业拆成多个文件。

### 6.2 JSON 格式要求

```json
[
  { "id": "hw1_judge_1", ... },
  { "id": "hw1_judge_2", ... },
  ...
]
```

- 顶层是数组 `[...]`
- UTF-8 编码（不要 BOM）
- 缩进 2 空格
- 不要尾逗号
- 所有字符串用双引号 `"`

### 6.3 中文字符处理

- 选项/题干中的中文引号 `""` **必须**替换为 `「」`，否则 JSON 解析会报错
- em-dash `——` 替换为 `——`
- 其他 Unicode 中文标点没问题

6. ❌ 判断题 `options` 不为 `null`
