# study-hub docx 题库规范 — LLM 生成提示词

复制以下内容给 LLM，生成的 docx 可直接用 `convert.js` 转换为 JSON 导入刷题系统。

---

## 提示词

```
请生成一个 docx 文件，内容格式严格遵循以下规范。文件名为「第X章_章节名.docx」（如「第一章_世界的物质性及其发展规律.docx」）。

【题型标记】
一、单项选择题
二、多项选择题
三、判断题

【题目格式】
每道题以「题目」二字开头，题干末尾用中文全角括号「（）」收尾。
选项用大写字母加中文全角点「．」(U+FF0E)，选项之间用空格分隔，不要换行。
选项文本中绝对不要出现「正确」「错误」字样——那是答案，不能泄露。

格式示例：
题目马克思主义认为，世界的真正统一性在于它的（）A．实践性 B．运动性 C．物质性 D．客观性

【解析格式 — 紧接选项之后，不要空格】
简略解析 + 逐选项判对错（这是给解析器提取正确答案用的，必须严格按此格式）：
简略解析A. 正确。[该选项为什么对] B. 错误。[该选项为什么错] C. 错误。[该选项为什么错] D. 错误。[该选项为什么错]

详细解释 + 一段话总结本题考点（这是给学生看的详细解析）：
详细解释[考点总结、知识背景、易错点提示等]

完整示例：
题目马克思主义认为，世界的真正统一性在于它的（）A．实践性 B．运动性 C．物质性 D．客观性简略解析A. 错误。实践性是马哲的显著特点，但并非世界统一性的依据。B. 错误。运动是物质的根本属性，不是世界统一性的基础。C. 正确。辩证唯物主义认为世界统一于物质。D. 错误。客观性是对物质的描述，不是统一性本身。详细解释本题考查世界的物质统一性原理。马克思主义哲学认为，世界真正的统一性在于它的物质性，这是辩证唯物主义最基本的原理。

【判断题格式】
判断题无选项，题干后直接跟解析。
题目社会主义社会的基本矛盾是对抗性的矛盾简略解析错误。社会主义社会的基本矛盾是非对抗性的，可以通过社会主义制度本身不断得到解决。详细解释社会主义基本矛盾仍然是生产关系与生产力、上层建筑与经济基础的矛盾，但性质是非对抗性的。

【多选题格式】
多选题选项和解析格式与单选题完全相同，正确答案为多个字母连写，解析器中会自动提取。
题目下列选项中属于马克思主义哲学基本特征的有（）A．实践性 B．科学性 C．革命性 D．阶级性简略解析A. 正确。实践性是马哲最显著的特征。B. 正确。马哲揭示了自然、社会和思维发展的普遍规律。C. 正确。马哲是批判和革命的学说。D. 正确。马哲公开申明为无产阶级服务。详细解释马克思主义哲学的基本特征包括：实践性、科学性、革命性和阶级性，它们是统一的整体。

【关键规则 — 必须遵守】
1. ✅ 选项字母用中文全角点「．」，不要用英文句号「.」
2. ✅ 解析中的字母用英文句号「.」
3. ✅ 每道题必须有简略解析和详细解释
4. ✅ 选项文本绝对不能包含「正确」或「错误」
5. ✅ 题干末尾统一用「（）」
6. ✅ 不要出现「2分」「0分」等分数标记
7. ✅ 题目之间自然衔接，无需换行分隔
```

## 解析器说明（给开发者参考）

上述提示词生成的 docx 会被 `convert.js` 按以下规则解析：

| docx 内容 | JSON 字段 | 答题页显示 |
|-----------|-----------|------------|
| `题目` + 题干 + `（）` | `question` | 题干 |
| `A．xxx B．xxx` | `options: {A, B, ...}` | 选项按钮 |
| `简略解析A. 正确...B. 错误...` | `explanation`（同时提取 `answer`） | 📝 **逐项分析**（显示在上，✓/✗ 格式化） |
| `详细解释...` | `detail` | 📖 **解析**（显示在下，考点总结） |
| 文件名 `第X章_XXX` | `homework` + `tags` | 作业分组 + 知识点标签 |

> **显示顺序**：答题后先出现 📝 逐项分析（每选项对错原因），再出现 📖 解析（整体考点总结）。两个区域都直接展开，无需点击折叠。
